# Retail Sales Performance Dashboard

## Overview
This project presents an interactive Tableau dashboard built using the Sample Superstore dataset. The dashboard provides insights into retail sales performance by analyzing sales trends, regional performance, customer segments, and top-performing products.

## Objectives
- Analyze sales performance across different product categories.
- Identify the highest-performing regions.
- Compare profit generated by customer segments.
- Track monthly sales trends.
- Identify the top 10 products by sales.

## Tools Used
- Tableau Public
- Microsoft Excel
- SQL
- Git & GitHub

## Dataset
- Sample Superstore Dataset

## Dashboard Features
- Sales by Category
- Monthly Sales Trend
- Sales by Region
- Profit by Segment
- Top 10 Products

## Key Insights
- Technology generated the highest sales among all categories.
- The West region achieved the highest overall sales.
- The Consumer segment generated the highest profit.
- Sales peaked during November.
- A small number of products contributed significantly to total revenue.
